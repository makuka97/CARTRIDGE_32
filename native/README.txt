CARTRIDGE32 — Native Emulator Binaries
=======================================

Place emulator folders here:

  native/
    nestopia/       ← nestopia.exe    (NES)
    snes9x/         ← snes9x.exe      (SNES)
    mgba/           ← mgba.exe        (GB/GBC/GBA)
    gens/           ← gens.exe        (Genesis/SMS)
    stella/         ← stella.exe      (Atari 2600)
    prosystem/      ← ProSystem.exe   (Atari 7800)
    mednafen/       ← mednafen.exe    (Lynx/TG16/NGP)
    vice/           ← x64.exe         (Commodore 64)
    mupen64plus/    ← mupen64plus.exe (N64)
    dosbox/         ← dosbox.exe      (MS-DOS)

Each folder should contain all files from the emulator's zip release.
CARTRIDGE32 will find the exe automatically.
